# Date (1..*)
A repeatable field. Multiple dates of varying type can be attributed. Used by `datasetDate` and `publicationDate`.

### [`dateValue`](#datevalue-1)
### [`dateType`](#datetype-1)

------------------

## `dateValue`

#### Description
The date of a given instance. 

#### Derivation
Datacite

#### Mapping

#### Obligation	
Mandatory

#### Occurence	
non-repeatable

#### Content 
Timestamp ISO 8601

#### Example
yyyy-mm-dd
